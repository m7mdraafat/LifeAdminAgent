## Creating Life Events & Checklists
When creating events with checklists:
1. Create MANY detailed, actionable tasks (minimum 25-50 for learning plans)
2. Each task must be SPECIFIC and MEASURABLE
3. Include real resources (books, websites, tools) with names
4. Give a SHORT confirmation response (don't list all tasks in chat!)

### CRITICAL: Learning/Training Plans Must Include:
- **Specific resources**: Book titles, course names, websites (e.g., "LeetCode", "Coursera: Learning How to Learn")
- **Concrete exercises**: "Solve 5 two-pointer problems" not "Practice problems"
- **Time estimates**: "2 hours", "30 min daily"
- **Measurable outcomes**: "Complete quiz with 80%+", "Build 3 projects"
- **Weekly breakdown**: For multi-month plans, create week-by-week tasks
- **Practice quantity**: "Solve 50 easy problems" not just "Practice"

## Sample: 3-Month Problem Solving Plan Structure (40+ tasks)
For reference when creating similar plans:
- **Foundations (Week 1)**: Define goals, gather 3-5 key resources, understand problem types
- **Week 2-4**: Learn 5 core techniques (5 Whys, Fishbone, SCAMPER, Mind Maps, First Principles) - 1 technique per 2-3 days with practice
- **Week 5-8**: Apply to 20+ practice problems (logic puzzles, case studies, real scenarios)
- **Week 9-10**: Advanced techniques (systems thinking, lateral thinking)
- **Week 11-12**: Real project application + reflection
- Each week should have 3-5 specific tasks with resources
